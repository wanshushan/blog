## 关于题目

[CF](https://codeforces.com/problemset/problem/1847/C) | [洛谷](https://www.luogu.com.cn/problem/CF1847C)

## 一些前置知识

:::tip
1.异或的交换律: a⊕b=b⊕a

2.异或的结合律：(a⊕b)⊕c=a⊕(b⊕c)

3.异或的特殊性质：a⊕a=0

:::

由上述知识我们可以得出一个重要结论

>[!NOTE]
>可以通过求异或前缀和得出异或子段和
>
> 如：(用ah表示前缀和)\
> a3^a4=ah4^ah2

## 简单思路

根据题意不难想到最大战斗力就是**序列的最大连续子段异或和**

一个简单的想法是求出所有的异或前缀和然后枚举两个端点找最大值

```cpp
for (int i = 0; i <= n; i++)
{
    for (int j = i + 1; j <= n; j++)
    {
        ans = max(ans, ah[i] ^ ah[j]);
    }
}
```

> [!WARNING]
> 很明显对于10^5的数据  绝对会**TLE**

## 小巧思

但是紧接着，我们观察到a<=2^8\
这意味着不同的a最多只有256个，不同的ah也是如此
> [!TIP]
> 我们完全可以将ah去重后再枚举！！！

```cpp
map<ll, ll> mp;
vector<ll> uah;
for (ll i = 0; i <= n;i++){
    if(i==0){
        ah[i] = 0;
    }
    else{
        ah[i] = ah[i - 1] ^ a[i];
    }
    if(!mp[ah[i]]){
        uah.push_back(ah[i]);
    }
    mp[ah[i]]++;
}
for(auto i:uah){
    for(auto j:uah){
        ans = max(ans, i ^ j);
    }
}
```

                ah[i] = 0;
            }

                ah[i] = ah[i - 1] ^ a[i];
            }
            if(!mp[ah[i]]){
                uah.push_back(ah[i]);
            }
            mp[ah[i]]++;
        }
        for(auto i:uah){
            for(auto j:uah){
                ans = max(ans, i ^ j);
            }
        }

    }
}
